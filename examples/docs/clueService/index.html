expressAPI.js

SER 421 Lab 7 Express Server API

The API acts as a server that manages a Clue activity by returning JSON packages of rooms, suspects and weapons.

Body-parser referenced from: https://github.com/kgary/ser421public/blob/master/NodeExpress/express_post.js

options

Defines the header for an options request from the browser. It defines the options and access permissions for the client and a wildcard CORS policy.

options(Endpoint: string, function: function)
Parameters
Endpoint (string) to mark middleware procedure for all requests to the server
function (function) Anonymous function to set header information

all

Sets the header for all response objects. The header allows for cross-origin reference from all hosts and defines the content interaction to use JSON as GET/POST parameters

all(Endpoint: string, function: function)
Parameters
Endpoint (string) to mark middleware procedure for all requests to the server
function (function) Anonymous function to set header information

all

Defines the generic endpoint for all other undefined URLs on the current host. It returns a 404 error.

all(null: string, function: function)
Parameters
null (string) '/*' Wildcard enpoint URL
function (function) Anonymous function to return the generated 404 error

get

Returns all lists (suspects, rooms, weapons) in one JSON.

get(null: string, function: function)
Parameters
null (string) '/' Root enpoint URL
function (function) Anonymous function to set initial layout

get

Defines the GET endpoint for the suspects operation. Suspects are returned in a JSON format and are a list of all suspects in the suspects list.

get(null: string, function: function)
Parameters
null (string) '/suspects' enpoint URL
function (function) Anonymous function to grab request/resource objects.
