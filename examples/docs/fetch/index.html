fetch.js

Uses Fetch to retrieve JSON packages from a defined API that returns rooms, suspects and weapons in a game of clue.

fetch.js
Version: 2018-12-01
Author: Kevin Shelley (krshelle@asu.edu)

displayActivity

Retrieves information from the server via its root endpoint and displays it on the fetch.html web page. See the server 'clueService.js' for more information about the root enpoint and the various responses enabled by the server.

displayActivity()

fetchClue

Sends a fetch request to the server to retrieve the necessary data. Fetch requests are made to the host at a specified endpoint. See the 'clueService.js' server for information about the different enpoints.

fetchClue(host: string, endpoint: string): Promise
Parameters
host (string) The host and port number to send the request to.
endpoint (string) The endpoint of the get request on the server.
Returns
Promise: A Promise with the retrieved JSON.

handleError

Displays an error message based on the request returned by the server. If no address is returned, then a 0 error is generated with a default error description.

handleError(error: JSON)
Parameters
error (JSON) The request object with the error information.
