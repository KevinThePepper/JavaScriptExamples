PromiseAsyncAwait.js

Displays the functionality of Promises and await/asynch method to create a synchronous environment on the asynchronous event loop.

PromiseAsyncAwait.js
Version: 12/1/2018
Author: Kevin Shelley (krshelle@asu.edu)

fs

Required for file IO. Used to create file directories.

fs

makeDir

Creates a directory with all users read/write/execute permissions. Permissions are granted to allow the user to delete the folder after executing this script.

makeDir(dir: string): any
Parameters
dir (string) The relative/absolute directory.
Returns
any: A Promise object to create the directory and modify the permissions.

createPromise

Displays using the returned Promise from the makeDir method using the Promise object syntax. Creates a 'promises/' folder with 3 sub-directories.

createPromise()

awaitAsynch

Displays using the returned Promise from the makeDir method using await/async. Creates an 'async/' folder with 3 sub-directories.

awaitAsynch()
