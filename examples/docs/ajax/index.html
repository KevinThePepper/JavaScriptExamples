ajax.js

Uses AJAX to retrieve JSON packages from a defined API that returns rooms, suspects and weapons in a game of clue.

ajax.js
Version: 2018-12-01
Author: Kevin Shelley (krshelle@asu.edu)

Type: string

displayActivity

Retrieves information from the server via its root endpoint and displays it on the ajax.html web page. See the server 'clueService.js' for more information about the root enpoint and the various responses enabled by the server.

displayActivity()

ajaxGet

Sends an AJAX GET request to the server specified by the host url and the parameter endpoint. If no information is retrieved or the server returns an error, the error will be handled by the appropriate handleError method.

ajaxGet(endpoint: string)
Parameters
endpoint (string) The endpoint of the get request to the server.

getRequestObject

Returns the AJAX object to enable async communication with the back-end server.

getRequestObject(): any
Returns
any: The AJAX request object if the window enables it, null if the window does not support AJAX

handleResponse

Sends request status to the debugger to analyze the state of the request.

handleResponse(request: JSON)
Parameters
request (JSON) The request object to the server.

handleError

Displays an error message based on the request returned by the server. If no address is returned, then a 0 error is generated with a default error description.

handleError(request: JSON)
Parameters
request (JSON) The request object with the error information.
